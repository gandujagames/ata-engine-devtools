# Roadmap

## Phase 1: Basic developer utilities

- Vulkan environment checker
- Engine log scanner
- Asset name validator
- CMake clean build helper

## Phase 2: Better validation

- Shader compile log parser
- Missing texture reference scanner
- Material naming convention checker
- Project folder structure validator

## Phase 3: CI and release workflow

- GitHub Actions validation
- Automated test examples
- Versioned releases
- Contribution guidelines

## Long-term goal

The long-term goal is to provide lightweight open-source tools for independent game developers and small engine teams who need practical workflow validation without heavy infrastructure.
